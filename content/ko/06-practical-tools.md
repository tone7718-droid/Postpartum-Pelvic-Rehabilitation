# 6장. 실천 도구 — 증상 일지와 6주 검진 체크리스트

> 이 장은 운동을 "꾸준히, 안전하게" 이어가도록 돕는 **기록·소통 도구**입니다.
> 기록은 회복 추세를 눈으로 보게 해주고, 진료·물리치료 때 **정확한 정보**를 전달하는 데 큰 도움이 됩니다.

---

## 6-1. 증상 일지 (하루 1분)

매일 자기 전 1분이면 충분합니다. 통증·요실금 같은 증상은 날마다 들쭉날쭉해서, 기록해 두면 **"전반적으로 나아지는지"**를 객관적으로 알 수 있습니다.

**기록 항목**
- **통증 위치·강도(0~10)**: 어디가(치골/엉치/허리/없음), 얼마나 아픈지
- **요실금**: 샌 횟수, 어떤 상황(기침·재채기·운동 등)
- **압박감/묵직함(탈출감)**: 있었는지, 저녁에 심해지는지
- **운동 수행**: 어떤 운동을 몇 회 했는지, 무리는 없었는지
- **복부 솟구침(coning)**: 어떤 동작에서 배가 솟았는지
- **특이사항**: 오로(출혈) 변화, 컨디션, 수면 등

**주간 기록표 예시 (복사해서 사용)**

| 날짜 | 통증(부위/0~10) | 요실금(횟수/상황) | 압박감 | 운동(내용/횟수) | coning | 메모 |
|---|---|---|---|---|---|---|
| / | | | | | | |
| / | | | | | | |
| / | | | | | | |
| / | | | | | | |
| / | | | | | | |
| / | | | | | | |
| / | | | | | | |

> 💡 **읽는 법**: 숫자가 **줄어드는 추세**면 잘 가고 있는 것. 같은 운동에서 통증·요실금·압박감이 **늘어나면** 한 단계 낮추고, 2주 이상 지속·악화되면 전문가 상담.

---

## 6-2. 6주(산후) 검진 — 미리 챙길 체크리스트

산후 6주 진료는 단순 진찰이 아니라 **운동을 다음 단계로 올려도 되는지 확인받는 관문**입니다. 짧은 진료 시간에 놓치지 않도록 미리 적어 가세요.

**진료 전, 적어 갈 것 (증상 일지에서 옮겨 쓰기)**
- [ ] 골반·치골·허리 **통증** 위치와 정도(언제 심해지는지)
- [ ] **요실금/변실금** 여부와 빈도
- [ ] 질 쪽 **묵직함·빠지는 느낌(탈출감)** 여부
- [ ] **복직근 이개** 자가 점검 결과(손가락 너비/탄탄함)
- [ ] 분만 관련 회복(회음부/제왕절개 상처) 불편 사항
- [ ] 성관계 시 통증, 기분 변화(산후 우울 관련) 등

**진료 때 물어볼 것**
- [ ] **지금 어떤 운동까지 해도 되나요?** (코어 강화·근력 운동 시작 가능 시점)
- [ ] **복직근 이개/골반저 상태**를 확인해 주실 수 있나요?
- [ ] **여성 골반 전문 물리치료(PFPT)** 의뢰가 필요할까요?
- [ ] 통증이 지속되는데 **골반 지지대(SI 벨트)**나 추가 검사(영상)가 필요할까요?
- [ ] 다음 단계로 넘어가도 되는 **나만의 기준/주의 신호**는?

**바로 의료진을 찾아야 하는 위험 신호(Red Flags) — 6주를 기다리지 말 것**
- 오로가 다시 빨개지거나 양이 늘어남 / 한 시간 만에 패드를 적시는 다량 출혈
- 한쪽 종아리가 붓고 아프고 빨갛고 뜨거움(혈전 의심)
- 38°C 이상 발열, 악취 나는 분비물(감염 의심)
- 점점 심해지는 골반·치골 통증으로 걷기·돌아눕기 곤란
- 갑작스러운 대소변 조절 상실, 질 안에 덩어리가 만져지는 느낌

> 자세한 평가 방법은 **[1장 셀프 평가](01-self-assessment.md)**, 위험 신호 전체 목록은 **[4장 4-2](04-faq-glossary.md)**를 참고하세요.

## 6-3. 함께하는 사람을 위한 팁 (배우자·가족)

- **함께 하기**: "오늘 호흡 운동 같이 해볼까?"처럼 부담 없이 같이 시작하면 꾸준해집니다.
- **무리 신호 살피기**: 통증을 호소하면 즉시 멈추게 하고 증상 일지에 적어 둡니다.
- **회복 우선 분위기**: 체중·몸매보다 **회복과 휴식·수면**을 먼저 챙기도록 돕고, 작은 진전을 함께 기뻐해 주세요.
- **집안일·육아 분담**으로 무거운 들기·수면 부족을 줄이는 것이 회복에 직접 도움이 됩니다.

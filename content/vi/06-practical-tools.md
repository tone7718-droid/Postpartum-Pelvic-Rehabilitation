# Chương 6. Công cụ thực hành — Nhật ký triệu chứng và danh sách kiểm tra khám tuần 6

> Chương này là **công cụ ghi chép·trao đổi** giúp duy trì tập luyện "đều đặn, an toàn".
> Ghi chép giúp nhìn thấy xu hướng hồi phục bằng mắt và truyền đạt **thông tin chính xác** khi khám·vật lý trị liệu.

---

## 6-1. Nhật ký triệu chứng (1 phút mỗi ngày)

Mỗi tối trước khi ngủ 1 phút là đủ. Triệu chứng như đau·són tiểu thay đổi thất thường từng ngày, nên ghi lại sẽ biết khách quan **"tổng thể có đang tốt lên không".**

**Mục ghi chép**
- **Vị trí·mức độ đau (0~10)**: ở đâu (xương mu/xương cùng/lưng/không), đau bao nhiêu
- **Són tiểu**: số lần rò, trong tình huống nào (ho·hắt hơi·tập…)
- **Cảm giác đè/nặng (cảm giác sa)**: có không, có nặng hơn vào buổi tối không
- **Tập luyện**: làm bài tập gì mấy lần, có quá sức không
- **Phình bụng (coning)**: ở động tác nào bụng phình lên
- **Ghi chú khác**: thay đổi sản dịch (ra máu), thể trạng, giấc ngủ…

**Bảng ghi tuần ví dụ (sao chép để dùng)**

| Ngày | Đau (vị trí/0~10) | Són tiểu (lần/tình huống) | Đè nặng | Tập (nội dung/lần) | coning | Ghi chú |
|---|---|---|---|---|---|---|
| / | | | | | | |
| / | | | | | | |
| / | | | | | | |
| / | | | | | | |
| / | | | | | | |
| / | | | | | | |
| / | | | | | | |

> 💡 **Cách đọc**: Nếu các con số **có xu hướng giảm** là đang đi đúng. Nếu cùng một bài tập mà đau·són tiểu·đè nặng **tăng lên** thì hạ một bậc; nếu kéo dài·nặng thêm trên 2 tuần thì hỏi chuyên gia.

## 6-2. Khám tuần 6 (sau sinh) — danh sách chuẩn bị trước

Khám tuần 6 không chỉ là thăm khám đơn thuần mà là **cửa ải để xác nhận có thể nâng bài tập lên bước tiếp theo hay không.** Hãy ghi trước để không bỏ sót trong thời gian khám ngắn ngủi.

**Trước khi khám, ghi sẵn (chép từ nhật ký triệu chứng)**
- [ ] Vị trí·mức độ **đau** chậu·xương mu·lưng (lúc nào nặng hơn)
- [ ] Có **són tiểu/són phân** và tần suất
- [ ] Cảm giác **nặng·như tụt (cảm giác sa)** ở vùng âm đạo
- [ ] Kết quả tự kiểm tra **tách cơ thẳng bụng** (bề rộng ngón tay/độ chắc)
- [ ] Khó chịu liên quan hồi phục sau sinh (vết tầng sinh môn/vết mổ)
- [ ] Đau khi quan hệ, thay đổi tâm trạng (liên quan trầm cảm sau sinh)…

**Hỏi khi khám**
- [ ] **Hiện tại có thể tập đến mức nào?** (thời điểm bắt đầu được bài tập cơ lõi·sức mạnh)
- [ ] Bác sĩ có thể kiểm tra **tình trạng tách cơ thẳng bụng/sàn chậu** giúp không?
- [ ] Có cần giới thiệu **vật lý trị liệu sàn chậu nữ (PFPT)** không?
- [ ] Đau kéo dài thì có cần **đai chậu (đai SI)** hoặc kiểm tra thêm (hình ảnh) không?
- [ ] **Tiêu chí/dấu hiệu cảnh báo riêng** để được phép sang bước tiếp theo là gì?

**Dấu hiệu cần đi khám ngay (Red Flags) — đừng chờ đến tuần 6**
- Sản dịch đỏ trở lại hoặc lượng tăng / ra máu nhiều ướt đẫm băng trong một giờ
- Một bên bắp chân sưng·đau·đỏ·nóng (nghi huyết khối)
- Sốt từ 38°C, dịch tiết hôi (nghi nhiễm trùng)
- Đau chậu·xương mu ngày càng nặng đến mức khó đi·khó trở mình
- Mất kiểm soát tiểu·tiện đột ngột, sờ thấy khối trong âm đạo

## 6-3. Mẹo cho người đồng hành (chồng·gia đình)

- **Cùng làm**: Bắt đầu cùng nhau nhẹ nhàng như "Hôm nay tập thở chung nhé?" sẽ giúp duy trì đều đặn.
- **Để ý dấu hiệu quá sức**: Khi vợ kêu đau thì cho dừng ngay và ghi vào nhật ký triệu chứng.
- **Tạo không khí ưu tiên hồi phục**: Giúp ưu tiên **hồi phục·nghỉ ngơi·giấc ngủ** hơn cân nặng·vóc dáng, và cùng vui với từng tiến bộ nhỏ.
- **Chia sẻ việc nhà·chăm con** để giảm nâng nặng·thiếu ngủ — điều này giúp ích trực tiếp cho hồi phục.
